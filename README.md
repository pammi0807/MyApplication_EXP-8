# MyApplication_EXP-8

Aim
To create an Android-based temperature converter application that performs conversion between Fahrenheit and Celsius scales.
Algorithm


1.Start Android Studio and create a mobile application project.


2.Create the layout file using LinearLayout with EditText, Buttons, and TextView components.


3.Provide an input field for entering temperature values.


4.Create buttons for converting temperature values and clearing data.


5.Initialize all widgets inside the onCreate() method.


6.Use setOnClickListener() for handling button click events.


7.When the Convert to Fahrenheit button is clicked, read the Celsius input value.


8.Apply the conversion formula:
F=9C5+32F = \frac{9C}{5} + 32F=59C​+32


9.Display the Fahrenheit result in the TextView.


10.When the Convert to Celsius button is clicked, read the Fahrenheit value.


11.Apply the conversion formula:
C=5(F−32)9C = \frac{5(F-32)}{9}C=95(F−32)​


12.Show the converted Celsius result on the screen.


13.Implement the Clear button to reset the EditText and result display.


14.Handle empty input conditions properly.


15.Execute the application and test all functionalities successfully.
